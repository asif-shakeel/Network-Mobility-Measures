# Network-Mobility-Measures
Code for the paper: Network-Level Measures of Mobility from Aggregated Origin-Destination Data (https://arxiv.org/abs/2502.04162v2)

code_h3 for H3 Resolution 6 analyses: data can be generated by the code in https://github.com/asif-shakeel/Mobility-Trajectories for the paper: Mobility Trajectories from Network-Driven Markov Dynamics (https://arxiv.org/pdf/2601.06020)

code_nm for Netmob2024 Data analyses: data is under restrictions as stated in the paper
